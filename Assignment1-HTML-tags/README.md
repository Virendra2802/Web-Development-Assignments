# HTML Text Formatting Tags Demo

This project demonstrates the usage of various HTML text formatting tags using a simple webpage. The page displays a table containing HTML formatting tags, their descriptions, and examples, along with a paragraph that applies these tags in real content.

The goal of this project is to help beginners understand how different HTML tags modify the appearance and meaning of text on a webpage.

---

## Project Overview

The webpage includes:

- A styled HTML table that lists common text formatting tags.
- A live example column showing how each tag appears in the browser.
- A descriptive paragraph that demonstrates multiple formatting tags in a real sentence structure.
- A preformatted text block to show how `<pre>` preserves spacing and line breaks.
- Basic CSS styling to improve readability and layout.

---

## Technologies Used

- HTML5 – Structure of the webpage  
- CSS3 – Styling and layout  

No JavaScript or external frameworks are used in this project.

## HTML Tags Demonstrated

### `<b>`
Displays text in bold without adding extra importance.

### `<strong>`
Displays text in bold and indicates strong importance in semantic HTML.

### `<i>`
Displays text in italic style.

### `<em>`
Represents emphasized text. Browsers usually display it in italic.

### `<u>`
Underlines the text.

### `<mark>`
Highlights text with a background color.

### `<small>`
Displays text in a smaller font size.

### `<del>`
Represents deleted text and displays it with a strike-through.

### `<ins>`
Represents inserted text and usually appears underlined.

### `<sub>`
Displays subscript text, commonly used in chemical formulas (Example: H₂O).

### `<sup>`
Displays superscript text, often used in mathematics (Example: 10²).

### `<code>`
Displays text in a monospace font used for programming code.

### `<pre>`
Displays preformatted text while preserving spaces and line breaks.
