# ☕ Destination Cafe - HTML Semantic Tags Project

A clean, single-page website for **Destination Cafe** built exclusively with HTML to demonstrate the use of text formatting and semantic elements. 

## 🎯 Project Objective
The goal of this project was to create a real-world cafe landing page while incorporating a specific set of HTML tags to handle text styling, pricing updates, and technical data display.

## 🛠️ HTML Tags Implemented
This project successfully utilizes the following tags:

| Tag | Usage in Project |
 :--- | :--- |
| `<b>` | Bolded the Cafe name and menu items. |
| `<strong>` | Emphasized the brand slogan. |
| `<i>` | Italicized the cafe name in the "Our Story" section. |
| `<em>` | Added emphasis to the mission statement. |
| `<u>` | Underlined the "Visit us today" call to action. |
| `<mark>` | Highlighted the Flash Sale announcement. |
| `<small>` | Used for "Customer Favorite" annotations. |
| `<del>` | Displayed the original (struck-through) menu prices. |
| `<ins>` | Displayed the new (underlined) discounted prices. |
| `<sub>` | Used for the subscript numbers in the caffeine chemical formula. |
| `<sup>` | Labeled new menu items with a "New" superscript badge. |
| `<code>` | Styled technical filenames within the menu. |
| `<pre>` | Maintained exact spacing for the operating hours and location layout. |

## 🚀 How to View
1. Clone this repository.
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge).
