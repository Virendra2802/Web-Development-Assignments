# ☕ Cafe Destination Website

## 📌 Overview
Cafe Destination is a modern and interactive cafe website that allows users to explore menu items, view gift cards, and add products to a cart. The website provides a smooth user experience with interactive UI elements like product popups, cart system, and payment section.

## 🚀 Features
- 🏠 Home Section – Attractive landing page with hero banner  
- 📖 About Section – Information about the cafe  
- 🍽️ Menu Section – Displays various food & beverage items  
- 🎁 Gift Cards Section – Promotional offers and rewards  
- 🛒 Cart System – Add items and view total price  
- 💳 Payment System – Select payment method and place order  
- 🔐 Login Button (UI) – Sign-in button for future integration  
- 📱 Responsive Layout – Works across devices  

## 🛠️ Technologies Used
- HTML5 – Structure of the website  
- CSS3 – Styling and layout  
- JavaScript – Interactivity and functionality
