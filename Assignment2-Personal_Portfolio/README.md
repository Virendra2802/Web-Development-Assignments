## ✨ Key Features

- **Modern Glassmorphism UI:** Semi-transparent navigation and contact elements with blur effects.
- **Dynamic Animations:** Integration of **AOS (Animate On Scroll)** for a premium user experience.
- **Typing Effect:** Interactive typewriter animation in the Hero section to grab attention.
- **Fully Responsive:** Optimized for all screen sizes (Mobile, Tablet, Desktop) using the **Bootstrap 5.3** grid system.
- **Dark Mode Aesthetic:** High-contrast design with neon accent colors (`--primary-color: #00fff2`).

## 🛠️ Technical Stack

- **Frontend:** HTML5, CSS3 (Custom Variables & Keyframe Animations), JavaScript
- **Frameworks/Libraries:** - [Bootstrap 5.3](https://getbootstrap.com/) (Layout & Components)
  - [AOS.js](https://michalsnik.github.io/aos/) (Scroll Animations)
  - [FontAwesome 6.4](https://fontawesome.com/) (Vector Icons)
  - [Google Fonts](https://fonts.google.com/) (Poppins)
