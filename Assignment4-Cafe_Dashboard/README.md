# ☕ Cafe Destination Website

## 📌 Overview
Cafe Destination is an interactive cafe website that allows users to explore menu items, purchase gift cards, manage a cart, and view business insights through a dashboard.

## 🚀 Features
- Home Section – Landing page  
- About Section – Cafe information  
- Menu Section – Interactive food items  
- Gift Cards Section – Offers and rewards  
- Dashboard Section – Charts using Chart.js  
- Cart System – Add items and view total  
- Payment System – UPI, Credit Card, Cash  
- Login Button (UI)

## 📊 Dashboard Features
- Sales Chart  
- Popular Items Chart  
- Built using Chart.js CDN  

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- Chart.js
